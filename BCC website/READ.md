# BCC Website Project

## Overview

This project is a simple multi-page school website for **BCC - Catholic College** using:

- HTML5
- CSS3
- JavaScript

The website includes:

- Home Page
- About Page
- Courses Page
- Teachers Page
- Department Page
- Contact Page

The project uses an **external CSS file** and an **external JavaScript file** for cleaner and organized code.

# How to Run the Project

## Method 1 — Open Directly

1. Open the project folder.
2. Double click `index.html`.

OR

3. Right click `index.html`
4. Select **Open with Browser**

---

## Method 2 — Using VS Code Live Server

### Step 1

Install:

- Visual Studio Code
- Live Server Extension

### Step 2

Open the project folder in VS Code.

### Step 3

Right click `index.html`

### Step 4

Click:

```plaintext
Open with Live Server
```
